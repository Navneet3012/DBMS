SQL LIKE Operator

The LIKE operator is used in a WHERE clause to search for a specified pattern in a column.

There are two wildcards often used in conjunction with the LIKE operator:

  • The percent sign (%) represents zero, one, or multiple characters
  • The underscore (_) represents a single character

The LIKE operator is case insensitive in some databases.

Syntax

SELECT column1, column2, ... FROM table_name WHERE columnN LIKE pattern;

Examples

Using the % Wildcard

Finds any values that start with "a":

SELECT * FROM Customers WHERE CustomerName LIKE 'j%';

Output

+--------------+---------+----------+------------+---------+ | CustomerName | Address | City | PostalCode | Country | +--------------+---------+----------+------------+---------+ | John | 21 Ce | Torronto | 4213 | Mexico | +--------------+---------+----------+------------+---------+

Finds any values that end with "a":

SELECT * FROM Customers WHERE CustomerName LIKE '%a';

Finds any values that have "or" in any position:

SELECT * FROM Customers WHERE CustomerName LIKE '%or%';

Using the _ Wildcard

Finds any values that have "r" in the second position:

SELECT * FROM Customers WHERE CustomerName LIKE '_r%';

Finds any values that start with "a" and are at least 3 characters in length:

SELECT * FROM Customers WHERE CustomerName LIKE 'a_%_%';

Finds any values that start with "a" and end with "o":

SELECT * FROM Customers WHERE CustomerName LIKE 'a%o';

Using NOT with LIKE

The NOT operator can be used to find records that do not match the pattern.

Finds any values that do not start with "a":

SELECT * FROM Customers WHERE CustomerName NOT LIKE 'j%';

Output

+--------------+-----------+-----------+------------+---------+ | CustomerName | Address | City | PostalCode | Country | +--------------+-----------+-----------+------------+---------+ | Cardinal | Skagen 21 | Stavanger | 4006 | Norway | | Cardinal | NULL | Stavanger | NULL | Norway | | Valen | NULL | New York | 5005 | USA | +--------------+-----------+-----------+------------+---------+
