SQL WHERE Clause

The WHERE clause is used to filter records. It is used to extract only those records that fulfill a specified condition.

Syntax

SELECT column1, column2, ... FROM table_name WHERE condition;

Note: The WHERE clause is not only used in SELECT statements, but also in UPDATE, DELETE, etc.

Example

Select all customers from Mexico:

SELECT * FROM Customers WHERE Country='Mexico';

This will select all records from the "Customers" table where the "Country" column has the value 'Mexico'.

Output

+--------------+---------+----------+------------+---------+ | CustomerName | Address | City | PostalCode | Country | +--------------+---------+----------+------------+---------+ | John | 21 Ce | Torronto | 4213 | Mexico | +--------------+---------+----------+------------+---------+

Text Fields vs. Numeric Fields

SQL requires single quotes around text values (most database systems will also allow double quotes). However, numeric fields should not be enclosed in quotes:

SELECT * FROM Customers WHERE PostalCode = 4006;

Output

+--------------+-----------+-----------+------------+---------+ | CustomerName | Address | City | PostalCode | Country | +--------------+-----------+-----------+------------+---------+ | Cardinal | Skagen 21 | Stavanger | 4006 | Norway | +--------------+-----------+-----------+------------+---------+

It is important to use the correct syntax to avoid errors.
