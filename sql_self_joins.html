SQL SELF JOIN

A SELF JOIN is a regular join where a table is joined with itself. It is useful for comparing rows within the same table, such as matching employees to their managers.

Syntax

SELECT column_names FROM table_name T1, table_name T2 WHERE condition;

SQL SELF JOIN Example

Match employees with their respective managers:

SELECT E.Name AS Employee, M.Name AS Manager FROM Employees E, Employees M WHERE E.ManagerId = M.Id;
