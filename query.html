SQL Top 3 Students Query

Description

In this query, we store students' marks and calculate the top 3 students based on performance.
Criteria: The student must have an aggregate above 60%, must not fail any subject (i.e., marks < 30), and must not be absent in any subject.

Example

  • Exactly 4 students failing overall
  • 2 students marked absent in one subject each
  • One absentee has the highest aggregate
  • Aggregate and percentage columns included
