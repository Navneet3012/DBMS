SQL Aggregate Functions

Aggregate functions perform calculations on a set of values and return a single result. They are commonly used with the GROUP BY clause in SELECT statements.

Common aggregate functions include:

  • MIN() – Returns the smallest value
  • MAX() – Returns the largest value
  • COUNT() – Counts matching rows
  • SUM() – Adds numeric column values
  • AVG() – Calculates average value

Note: All except COUNT() ignore NULL values.

COUNT() Function

Returns the number of rows that match the specified condition.

Syntax

SELECT COUNT(column_name) FROM table_name WHERE condition;

Example

Find the number of customers:

SELECT COUNT(CustomerID) FROM Products;

Note: NULL values are excluded.

Output

+-------------------+ | count(customerID) | +-------------------+ | 6 | +-------------------+

AVG() Function

Returns the average value of a numeric column.

Syntax

SELECT AVG(column_name) FROM table_name WHERE condition;

Example

Average price of all products:

SELECT AVG(price) FROM Products;

Output

+---------------+ | AVG(Price) | +---------------+ | 50.55 | +---------------+

SUM() Function

Returns the total of all values in a numeric column.

Syntax

SELECT SUM(column_name) FROM table_name WHERE condition;

Example

Sum of all quantities:

SELECT SUM(Quantity) FROM OrderDetails;

Output

+---------------+ | sum(quantity) | +---------------+ | 53 | +---------------+

Sum of Quantity for ProductID 11:

SELECT SUM(Quantity) FROM OrderDetails WHERE ProductID = 11;

Output

+---------------+ | sum(quantity) | +---------------+ | 10 | +---------------+

MIN() and MAX() Functions

MIN() returns the smallest value, and MAX() returns the largest value from the selected column.

Syntax

MIN:

SELECT MIN(column_name) FROM table_name WHERE condition;

MAX:

SELECT MAX(column_name) FROM table_name WHERE condition;

Example

Lowest quantity:

SELECT MIN(Quantity) FROM Products;

Output

+---------------+ | min(quantity) | +---------------+ | 4 | +---------------+

Highest quantity:

SELECT MAX(Quantity) FROM Products;

Output

+---------------+ | max(quantity) | +---------------+ | 15 | +---------------+
